# 🎓 EduAssist AI – Smart Student Support Chatbot

## 📌 Project Overview
EduAssist AI is a smart student support chatbot designed to help users with educational and technical queries.  
The chatbot can answer questions related to programming, AI, software development, exams, projects and general knowledge.

It is built using Python Flask and provides a clean modern web interface with chat history support.

---

## 🚀 Features

- 🤖 AI-style chatbot interface
- 💬 Chat history support
- 🌐 Web-based application
- 📚 Answers educational & technical questions
- 🧠 Smart keyword + Wikipedia support
- 🎨 Modern dark UI design
- 📱 Responsive layout

---

## 🛠️ Technologies Used

- Python
- Flask
- HTML
- CSS
- Wikipedia API
- Gunicorn

---

## 💡 Use Cases

- Student learning support
- Programming guidance
- AI & technical query solving
- Educational assistance
