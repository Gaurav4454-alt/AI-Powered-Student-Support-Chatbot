# 🎓 EduAssist AI – Smart Student Support Chatbot

## 📌 Project Overview

EduAssist AI is a smart student support chatbot designed to help users with educational and technical queries. The chatbot can answer questions related to programming, Artificial Intelligence, software development, exams, projects, and general knowledge.

Built using Python Flask, the application provides a clean and modern web interface with chat history support and intelligent response generation using keyword matching and Wikipedia integration.
